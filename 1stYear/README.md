# Contains work from the first year of the Computer Science Program

## Information about this folder
This folder holds the projects that I made during my first year at Cégep Heritage College. 
You will see: 
- My version of a casino dice game made with HTML, CSS & JavaScript.
- My problem solving reflection shows how computer graphics algorithms, specifically DDA and Bresenham’s line algorithms, are essential for creating high-quality visuals in fields like video game animation.
- A static website project for a fictional bagel shop called Somers Bagels, created as part of a web programming course using HTML and CSS.
- My version of sudoku made with Java.
- A Web Programming lab about different types of tables made with HTML styled with CSS.
